projects
Atom Identity Optimization in Molecular Frameworks
As part of the Digital Chemistry Laboratory at ETH Zurich, I worked on finding atom identities in molecular frameworks, that optimize a target property, e.g. the HOMO-LUMO Gap, using Mixed integer nonlinear programming (MINLP) and semidefinite programming (SDP)
FART: Flavor Analysis and Recognition Transformer Model
FART is a tool to predict the taste of a molecule, based on its SMILES representation. The model utilizes the fine-tuned LLM ChemBERTa. The FART dataset combines several sources into a FAIR dataset. Find our preprint here.
TpT Informer
Analytics and Management toolkit for TpT sellers.
As of March 2023, over 2500 Chrome extension users. Built with Next.js & Firebase.
Supported by a ETH Juniors Grant.
Squeeze
In a team of 4, we built a microfluidic device that exploits viscoelastic fluids to squeeze cells, allowing it to be used for transfection.
Simulated and optimized the device using COMSOL. Final report can be found here.
Connect 4 MCTS
A simple Monte Carlo Tree Search (MCTS) for Connect 4. Play against it here.
